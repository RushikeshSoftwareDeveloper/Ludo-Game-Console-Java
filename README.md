# Ludo-Game-Console-Java
Console-based Ludo Game developed in Java featuring dice rolling, two-player gameplay, and board movement logic.
# 🎲 Ludo Game (Console Version - Java)

## 📌 Project Description
A simple console-based Ludo game developed using Core Java.  
The game supports two players where each player rolls a dice and moves on the board.  
If a player rolls a 6, they get an extra turn. The first player to reach position 30 wins the game.

## 🚀 Features
- Dice roll (1–6)
- Two-player gameplay
- Board position tracking
- Extra turn on rolling 6
- Win condition at position 30

## 🛠 Concepts Used
✔ Random Class  
✔ Loops (while loop)  
✔ Conditional Statements  
✔ Methods  
✔ Object-Oriented Programming Basics

## 🎮 Game Rules
- Each player rolls a dice.
- Player moves according to dice value.
- Rolling a 6 gives an extra turn.
- First player to reach position **30** wins.

## ▶ How to Run
1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Run the main class
